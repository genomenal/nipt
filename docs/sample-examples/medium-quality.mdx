# Пример образца со средним качеством данных

Данными со средним качеством считаются те, к которым имеются некритичные замечания: покрытие < 0,01X и/или
несоответствие всем критериям качества после очистки. Для образца со средним качеством данных:
- все включенные стадии пайплайна будут отмечены как завершенные, однако некоторые метрики
оценки качества прочтений или выравнивания могут не удовлетворять критериям
(значок <img src={require('/img/rus/44-warning-status.png').default} width="28"/> в статусе соответствующих стадий):
