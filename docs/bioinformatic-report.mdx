# Биоинформатический отчёт

Вкладка "Биоинформатический отчёт" на странице образца включает основные метрики, которые помогут оценить
качество исходных данных, качество выравнивания и покрытие данных.

Разделы биоинформатического отчёта:
1. [Статистика файла](/bioinformatic-report#file-statistics).
2. [Отчёт о качестве](/bioinformatic-report#quality-report).
3. [Отчёт по выравниванию](/bioinformatic-report#alignment-report) - у образцов,
для которых стадия "Выравнивание" была включена в пайплайн и завершилась успешно.
4. [Отчёт по покрытию](/bioinformatic-report#coverage-report) - у образцов,
для которых стадия "Выравнивание" была включена в пайплайн и завершилась успешно.

### Статистика файла {#file-statistics}

Отчёт содержит основные показатели статистики прочтений в файле образца:
- **Всего прочтений** - количество прочтений в файле образца, выявленных на этапе проверки.
- **Минимальная длина прочтений**.
- **Максимальная длина прочтений**.
- **Медиана длины прочтений** - среднее значение длины прочтения в файле образца.

### Отчёт о качестве {#quality-report}

Отчёт содержит метрики качества прочтений, рассчитываемые
на стадии “Проверка качества”. Подробный отчёт по метрикам с визуализацией можно
скачать [на вкладке "Детали процесса анализа" в соответствующем разделе](/workflow-details/check-quality-and-cleanup).

### Отчёт по выравниванию {#alignment-report}

:::note На заметку
Раздел есть только у образцов, для которых [выравнивание](/settings/settings/alignment-settings) было включено в
пайплайн и успешно завершилось.
:::

Раздел включает статистику выравнивания и метрики, описывающие качество выравнивания. При большом количестве
ошибок секвенирования, неполных данных, высоком уровне контаминации или неверном определении источника (организма)
образца метрики укажут на возможную проблему.

1. **Статистика выравнивания**:
- **Всего прочтений** - суммарное количество прочтений в файле выравнивания.
- **Картированные прочтения** - количество прочтений образца, которые были картированы на референсный геном.
В скобках - доля количества таких прочтений от суммарного количества прочтений в файле (в процентах).
- **Некартированные прочтения** - количество прочтений образца, которые не удалось картировать на референсный геном.
В скобках - доля количества таких прочтений от суммарного количества прочтений в файле (в процентах).
- **Прочтения с множественным картированием** - количество прочтений, картированных на референсный геном несколько раз.
В скобках - доля количества таких прочтений от суммарного количества прочтений в файле (в процентах).
- **Прочтения с прямой цепи** - количество прочтений с прямой цепи, картированных на референсный геном.
В скобках - доля количества таких прочтений от суммарного количества прочтений в файле (в процентах).
- **Прочтения с обратной цепи** - количество прочтений с обратной цепи, картированных на референсный геном.
В скобках - доля количества таких прочтений от суммарного количества прочтений в файле (в процентах).
В нормальных данных количество прочтений с прямой и с обратной цепей одинаковое; разница в их количестве говорит о
том, что в данных есть транслокации ДНК (например, инверсии).
- **Парные прочтения** - суммарное количество парных прочтений в файле выравнивания.
В скобках - доля количества таких прочтений от суммарного количества прочтений в файле (в процентах).
- **Картированные парные прочтения** - количество парных прочтений образца, которые были картированы на
референсный геном.
В скобках - доля количества таких прочтений от суммарного количества прочтений в файле (в процентах).
- **Правильно картированные парные прочтения** - количество парных прочтений образца, правильно картированных на
референсный геном.
В скобках - доля количества таких прочтений от суммарного количества прочтений в файле (в процентах).
- **Всего выравниваний** - суммарное количество выравниваний прочтений образца на референсный геном.
- **Выравнивания с множественным картированием** - количество выравниваний одного прочтений на референсный геном
несколько раз.
В скобках - доля количества таких выравниваний от суммарного количества выравниваний в файле (в процентах).
- **Выравнивания с прямой цепи** - количество выравниваний прочтений с прямой цепи на референсный геном.
В скобках - доля количества таких выравниваний от суммарного количества выравниваний в файле (в процентах).
- **Выравнивания с обратной цепи** - количество выравниваний прочтений с обратной цепи на референсный геном.
В скобках - доля количества таких выравниваний от суммарного количества выравниваний в файле (в процентах).

### Отчёт по покрытию {#coverage-report}

:::note На заметку
Раздел есть только у образцов, для которых [выявление вариации числа копий](/settings/settings/cnv-discovery-settings)
было включено в
пайплайн, а стадия [вычисления покрытия](/workflow-details/calculate-coverage) успешно завершилась.
:::

Отчёт по покрытию генома данными позволяет оценить, насколько информативен анализ этих данных в целом.
Отчёт включает метрику **Coverage per nucleotide** - понуклеотидное покрытие на геноме.
Порог метрики - &ge; 0.1X (значение по умолчанию, может быть изменено
в [предустановке настроек](/settings/settings/alignment-settings#change-metric-thresholds),
используемой во время загрузки образцов). Если метрика
удовлетворяет порогу качества, она отмечена <img src={require('/img/rus/default_setting.png').default} width="27"/>,
а если не удовлетворяет, то <img src={require('/img/rus/45-warning-status.png').default} width="30"/>.

## Экспорт отчёта {#export}

Биоинформатический отчёт можно скачать в формате PDF.
Для этого нажмите на кнопку <img src={require('/img/results/export_pdf.png').default} width="150"/> в
правом верхем углу страницы отчёта.
