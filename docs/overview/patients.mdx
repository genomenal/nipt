# Пациенты

Блок "*Пациенты*" включает следующие страницы:
- "***Пациенты***" - содержит всех добавленных в систему и неархивированных пациентов и их данные.<br></br>

Если у вас ещё нет добавленных пациентов, то страница "*Пациенты*" выглядит так:

<p align="center">
<img src={require('/img/rus/no_patients_page.png').default} width="1000"/>
</p>

Страница с добавленными пациентами выглядит следующим образом:

<p align="center">
<img src={require('/img/rus/1-all-patients-page.png').default} width="1000"/>
</p>

Наверху страницы находятся следующие
кнопки: <img src={require('/img/rus/upload_samples_btn.png').default} width="150"/>, при нажатии на которую
открывается <a href="/uploading-samples/upload-samples-page">страница множественной загрузки данных</a>,
и <img src={require('/img/rus/add_patient_btn.png').default} width="150"/>,
открывающая окно <a href="/add-patient">создания пациента</a>. Кроме того, при наведении курсора на строку
пациента появляются следующие активные
элементы: <img src={require('/img/rus/upload_in_patient.png').default} width="19"/> позволяет <a href="/nipt/uploading-samples/additional-upload-for-patient">загрузить данные пациента</a>, <img src={require('/img/rus/forbid_edit_on_patients_page.png').default} width="19"/> - <a href="/nipt/add-patient#forbid-edit">запретить редактирование данных пациента</a>,
а <img src={require('/img/rus/delete_btn.png').default} width="20"/> -
архивировать данные пациента.
В строке пациента с запретом редактирования данных будет только кнопка для разрешения
редактирования данных: <img src={require('/img/rus/allow_edit_on_patients_page.png').default} width="20"/>.

- “***Архив***” - содержит тех пациентов, данные которых были архивированы. Вы можете перенести сюда
неактуальные данные или данные, которые вы хотите удалить. Пациенты, находящиеся в "*Архиве*", не отображаются
на странице "*Пациенты*".

<p align="center">
<img src={require('/img/rus/archive.png').default} width="1000"/>
</p>

Вверху страниц блока находится окно поиска, позволяющее искать пациентов по коду или имени:
<p align="center"><img src={require('/img/rus/search_patient.png').default} width="450"/></p>

Внизу страниц блока находятся:
- Информация о количестве пациентов: порядок пациентов, отображённых на странице, (максимум 10 пациентов может
быть показано на странице, так что порядок будет таким: 1-10, 11-20 и т.д.) и суммарное количество пациентов.
- Компонент пагинации: <img src={require('/img/rus/pagination.png').default} width="400"/>, который позволяет
перемещаться между страницами с пациентами. Номер открытой страницы выделен синим кружком. Если нажать на этот
номер, то страница перезагрузится. Чтобы переместиться на страницу с конкретным номером, либо нажмите на этот номер,
либо нажмите на <img src={require('/img/rus/three_points.png').default} width="30"/>, введите нужный номер и нажмите
клавишу ввода или кликните вне области заполнения значения.
Чтобы переместиться на предыдущую по номеру страницу,
нажмите на <img src={require('/img/rus/back.png').default} width="30"/>, а чтобы открыть следующую -
нажмите на <img src={require('/img/rus/next.png').default} width="30"/>.

### Колонки {#patients-columns}

На страницах блока "*Пациенты*" информация представлена в виде таблицы со следующими колонками:
1. **ПАЦИЕНТ** - информация о созданных пациентах (код, имя, фамилия пациента).
Если данные пациента запрещено редактировать, то слева от информации о пациенте будет отображена
иконка <img src={require('/img/rus/patient_with_no_edit.png').default} width="22"/>.
2. **СТАТУС** - статусы образцов каждого
пациента:<br></br><img src={require('/img/rus/in_process_status.png').default} width="22"/> - образец
обрабатывается;<br></br><img src={require('/img/rus/queed_status.png').default} width="14"/> - образец
стоит в очереди
на обработку;<br></br><img src={require('/img/rus/success_status.png').default} width="22"/> - образец
успешно обработан;<br></br><img src={require('/img/rus/problems_status.png').default} width="24"/> - образец успешно
обработан, но не удовлетворяет критериям
качества для проведения НИПС;<br></br><img src={require('/img/rus/failed_status.png').default} width="20"/> - образец
обработан с ошибкой;
3. **СОЗДАН** - дата добавления пациента в формате д.М.гггг;
4. **ИЗМЕНЕН** - дата последних изменений в данных пациента в формате д.М.гггг.

Нажав на строку пациента, вы откроете [его страницу](/add-patient#patient-page).

### Сортировка {#patients-sorting}

По умолчанию таблица отсортирована от пациентов, добавленных последними, к пациентам, добавленным первыми,
(т.е. по убыванию значений в колонке "*Создан*"). Таблицу можно отсортировать по возрастанию или убыванию
значений следующих колонок: "*Пациент*", "*Создан*", "*Изменен*".
Для сортировки таблицы по значениям колонки нажмите на её название (нажмите один раз, чтобы отсортировать
по возрастанию, или два раза, чтобы отсортировать по убыванию). После этого у названия колонки появится
стрелка <img src={require('/img/rus/sort_asc.png').default} width="17"/> (при сортировке по возрастанию)
или <img src={require('/img/rus/sort_desc.png').default} width="17"/> (при сортировке по убыванию).
Для сброса сортировки обновите страницу браузера. Колонка "*Пациент*" сортируется по алфавиту,
а колонки "*Создан*" и "*Изменен*" - от наиболее ранней к наиболее поздней дате и наоборот.

### Архивация, восстановление и удаление данных пациентов {#delete-patient}

Для ***архивации данных пациента*** и его образцов наведите курсор на строку пациента на странице "*Пациенты*" и
нажмите на появившийся активный элемент <img src={require('/img/rus/delete_btn.png').default} width="18"/>.
Пациент будет перемещен на страницу "*Архив*". Перейдя на эту страницу, вы можете:

1. ***Восстановить данные пациента***, наведя курсор на строку пациента и нажав на появившийся активный
элемент <img src={require('/img/rus/restore_btn.png').default} width="22"/>.
Тогда пациент будет перемещен обратно на страницу "*Пациенты*".

2. ***Удалить данные пациента*** и его образцы из системы, наведя курсор на строку пациента и нажав на
появившийся активный элемент <img src={require('/img/rus/delete_btn.png').default} width="18"/>.
Вы увидите окно подтверждения удаления пациента:

<p align="center">
<img src={require('/img/rus/delete_patient_confirmation.png').default} width="380"/>
</p>

Для подтверждения удаления пациента нажмите <img src={require('/img/runs/confirm.png').default} width="150"/>.
Для отмены удаления пациента нажмите <img src={require('/img/report_templates/cancel.png').default} width="150"/>
или вне окна подтверждения.

Кроме того, вы можете удалить все архивированные данные пациентов,
нажав на <img src={require('/img/rus/clear_archive_btn.png').default} width="170"/> на странице "*Архив*".
