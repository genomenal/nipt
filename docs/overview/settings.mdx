# Настройки

Блок "Настройки" включает страницы ["Настройки"](/overview/settings#settings-page) и
["Шаблоны отчётов"](/overview/settings#report-templates).

### Страница "Настройки" {#settings-page}

Страница содержит предварительные установки настроек для анализа образцов. Предустановки настроек
подробно описаны в [соответствущем разделе](/settings/settings/settings).

<p align="center">
<img src={require('/img/settings/settings_page.png').default} width="900"/>
</p>

:::note На заметку
Количество и вид предустановок настроек зависит от первоначальных настроек, сделанных администратором Системы.
:::

Вверху на странице находится кнопка <img src={require('/img/rus/add_setting.png').default} width="170"/>,
при нажатии на которую открывается окно [создания предустановки настроек](/settings/settings/settings#create-preset).

Информация на странице представлена в виде таблицы со следующими колонками:
1. **ПРЕДВАРИТЕЛЬНАЯ УСТАНОВКА НАСТРОЕК** - имя предустановки настроек;
2. **ОПИСАНИЕ** - описание предустановки настроек;
3. **ПО УМОЛЧАНИЮ** - колонка, в которой
галочкой <img src={require('/img/rus/default_setting.png').default} width="20"/>
отмечена та предустановка настроек, которая выбрана по умолчанию для анализа образцов;
4. **ИЗМЕНЕН** - дата последних изменений предустановки настроек в формате д.М.гггг.

Таблица отсортирована по имени предварительной установки настроек.

При наведении курсора на строку предустановки настроек, не выбранной по умолчанию, появляются следующие активные
элементы: <img src={require('/img/rus/set_as_default.png').default} width="22"/> позволяет установить
предустановку по умолчанию, а <img src={require('/img/rus/delete_btn.png').default} width="20"/> -
удалить предустановку.

### Шаблоны отчётов {#report-templates}

Страница содержит шаблоны отчётов, сформированные для представления результатов
анализа образцов в виде отчётов. Шаблоны отчётов подробно описаны
в [соответствущем разделе](/settings/report-templates/report-templates).

<p align="center">
<img src={require('/img/report_templates/rt_page.png').default} width="900"/>
</p>

:::note На заметку
Количество и вид шаблонов зависит от первоначальных настроек, сделанных администратором Системы.
:::

Вверху на странице находятся
следующие кнопки: <img src={require('/img/rus/template_import.png').default} width="170"/>,
при нажатии на которую открывается
окно <a href="/settings/report-templates/report-templates#import-template">импорта шаблона отчёта</a>,
и <img src={require('/img/rus/add_template.png').default} width="170"/>, открывающая страницу
[создания шаблона](/settings/report-templates/report-templates#add-template).

Информация на странице представлена в виде таблицы со следующими колонками:
1. **ИМЯ** - имя отчёта (дает название вкладке с отчётом на [странице образца](/results/results));
2. **ОПИСАНИЕ** - описание шаблона отчёта;
3. **АКТИВЕН** - колонка, в которой галочкой <img src={require('/img/rus/default_setting.png').default} width="20"/>
отмечены активные отчёты;
4. **ИЗМЕНЕН** - дата последних изменений шаблона отчёта в формате д.М.гггг.

Таблица отсортирована по именам шаблонов отчётов.

При наведении курсора на строку шаблона отчёта появляется
активный элемент <img src={require('/img/rus/delete_btn.png').default} width="20"/>, позволяющий
удалить шаблон.
