# Настройки

Блок "Настройки" доступен только для администратора Программы.
Он включает страницы ["Настройки"](/overview/settings#settings-page) и
["Шаблоны отчётов"](/overview/settings#report-templates).

### Страница "Настройки" {#settings-page}

Страница содержит предварительные установки настроек для анализа образцов. Предустановки настроек, добавленные
администратором, доступны всем пользователям Программы. Предустановки настроек
подробно описаны в [соответствующем разделе](/settings/settings/settings).

<p align="center">
<img src={require('/img/settings/settings_page.png').default} width="900"/>
</p>

Вверху на странице находится кнопка <img src={require('/img/rus/add_setting.png').default} width="170"/>,
при нажатии на которую открывается окно [создания предустановки настроек](/settings/settings/settings#create-preset).

Информация на странице представлена в виде таблицы со следующими колонками:
1. **ПРЕДВАРИТЕЛЬНАЯ УСТАНОВКА НАСТРОЕК** - имя предустановки настроек;
2. **ОПИСАНИЕ** - описание предустановки настроек;
3. **ПО УМОЛЧАНИЮ** - колонка, в которой
галочкой <img src={require('/img/rus/default_setting.png').default} width="20"/>
отмечена та предустановка настроек, которая выбрана по умолчанию для анализа образцов;
4. **ИЗМЕНЕН** - дата последних изменений предустановки настроек в формате д.М.гггг.

Таблица отсортирована по имени предварительной установки настроек.

При наведении курсора на строку предустановки настроек, не выбранной по умолчанию, появляются следующие активные
элементы: <img src={require('/img/rus/set_as_default.png').default} width="22"/> позволяет установить
предустановку по умолчанию, а <img src={require('/img/rus/delete_btn.png').default} width="20"/> -
удалить предустановку.

### Шаблон отчёта {#report-templates}

Страница содержит шаблон, сформированный для представления результатов
неинвазивного пренатального тестирования образцов в виде отчёта. [Отчёт на основе этого шаблона](/nipt-report)
строится для образцов всех пользователей Программы. Изменения в шаблон может вносить только администратор.

<p align="center">
<img src={require('/img/report_templates/rt_page.png').default} width="900"/>
</p>

Шаблон отчёта подробно описан в [соответствующем разделе](/settings/report-template).
