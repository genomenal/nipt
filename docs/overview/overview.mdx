# Знакомство с NIPT helicon

import Login from '/docs/imported_components/_login.mdx';

<Login />

Войдя в систему, вы попадете на главную страницу NIPT helicon - **["Образцы"](/overview/samples)**.
Если вы не пользовались NIPT helicon раньше и у вас нет загруженных образцов, то страница будет выглядеть следующим
образом:

<p align="center">
<img src={require('/img/rus/empty_sample_page.png').default} width="1000"/>
</p>

Впоследствии вы увидите здесь все образцы, находящиеся в этом аккаунте и принадлежащие неархивированным пациентам
и/или запускам:

<p align="center">
<img src={require('/img/rus/samples_page.png').default} width="1000"/>
</p>

Слева на странице, на боковой панели навигации представлены все разделы NIPT helicon (для перехода в
необходимый раздел нажмите левой кнопкой мыши на соответствущее название страницы):
1. Начальная страница ["Образцы"](/overview/samples);
2. Блок страниц ["Пациенты"](/overview/patients);
3. Блок страниц ["Запуски"](/overview/runs);
4. Блок страниц ["Настройки"](/overview/settings);
5. Блок страниц ["Администрирование"](/overview/administration) (есть только у администратора).

:::caution На заметку
Отображение блоков в составе панели навигации можно отрегулировать
на [странице "*Настройки пользователя*"](/settings/profile#navigation-panel).
:::

## Меню пользователя {#user-menu}

Внизу, на боковой панели навигации обозначен логин пользователя, под которым был осуществлен вход в Систему.
Наведя курсор мыши на логин и нажав на
появивишиеся три точки <img src={require('/img/rus/profile_options.png').default} height="20"/>, вы увидите
меню пользователя:

<p align="center">
<img src={require('/img/rus/user_options.png').default} width="210"/>
</p>

- "***Профиль***" - открывает страницу ["Настройки пользователя"](/settings/profile), где
находятся ваши персональные настройки аккаунта (email, логин, пароль) и настройки панели
навигации Программы.

- "***Сообщить о проблеме***" - открывает окно для отправки информации о возникшей у вас проблеме
администратору Системы:

<p align="center">
<img src={require('/img/rus/error_window.png').default} width="600"/>
</p>

- "***О программе***" - открывает окно со следующей информацией о Программе:
    - название программного обеспечения;
    - номер технических условий;
    - версия программного обеспечения;
    - сведения о разработчике и производителе программного обеспечения;
    - номер и дата регистрационного удостоверения.
- "***Выйти***" - осуществляет выход пользователя из Системы.
