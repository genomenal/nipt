# Основные шаблоны отчётов

В данном разделе описан основной шаблон отчёта, который используется для представления результатов анализа.
Скорее всего, он был добавлен в ваш профиль администратором при регистрации в Системе. Если у вас нет такого
шаблона, вы можете обратиться к администратору или добавить его самостоятельно, как описано ниже.

## Отчет НИПТ {#nipt-report}

- медицинский отчёт с результатами неинвазивного пренатального скрининга NIPT Helicon образца;
- влияет на построение отчета НИПТ.

### Условия построения отчёта {#glioma-report-conditions}

Отчет НИПТ строится для образца, если выполняются следующие условия:
1. Анализ образца успешно завершился (то есть все стадии анализа имеют статус "Завершено").
2. Для образца успешно завершилась стадия анализа "Выявление вариации числа копий".
3. Шаблон отчёта активен.
4. Шаблон отчёта был добавлен раньше, чем был обработан образец.

### Добавление шаблона отчёта НИПТ {#add-glioma-report}

Чтобы добавить шаблон отчёта НИПТ в систему, вы можете либо импортировать его,
либо создать самостоятельно.

#### Импорт шаблона отчёта для анализа глиомы {#import-glioma-report}

Скачайте файл шаблона (<a target="_blank" href="/reports/Отчёт для анализа глиомы.json" download="Отчёт для анализа глиомы.json">Отчёт для анализа глиомы</a>)
и импортируйте его, как описано [здесь](/settings/report-templates/report-templates#import-template).

#### Создание шаблона отчёта для анализа глиомы {#create-glioma-report}

Чтобы создать шаблон самостоятельно, выполните следующие действия:

1. На странице "Шаблоны отчётов" нажмите <img src={require('/img/rus/add_template.png').default} width="170"/>,
чтобы открыть страницу добавления шаблона отчёта.

2. На открывшейся странице заполните следующие поля:

- Имя: Отчёт для анализа глиомы
- Применим к образцам: Опухолевые
- Активен: да

3. Нажмите <img src={require('/img/report_templates/save_temaplate.png').default} width="140"/>, чтобы сохранить шаблон.

4. Добавьте в шаблон все блоки, перечисленные ниже. Список блоков приведен в том порядке, в котором они идут в
шаблоне. Если никаких уточнений после названия блока нет, значит используется блок, формируемый по умолчанию.

#### Блоки шаблона {#glioma-report-blocks}

1. "Информация о пациенте".
2. "Раздел".

Имя:
```
ОТЧЁТ ПО СОМАТИЧЕСКИМ МУТАЦИЯМ
```
Описание:
```
Образец опухоли
```

5. "Глиома: варианты уровня II".
6. "Маркеры иммунотерапии".
7. "Глиома: предполагаемый диагноз".
8. "Глиома: предлагаемые препараты".
9. "Глиома: подробности о мутациях".
10. "Глиома: варианты уровня III".
