# Шаблон отчета НИПТ

В данном разделе описан шаблон [отчета НИПТ](/nipt-report) - медицинского отчета,
который используется для представления результатов неинвазивного пренатального ДНК-скрининга образцов.
Скорее всего, он был добавлен в ваш профиль администратором при регистрации в Системе. Если у вас нет такого
шаблона, вы можете обратиться к администратору или добавить его самостоятельно, как описано ниже.

### Условия построения отчета {#report-conditions}

Отчет НИПТ строится для образца по описываемому шаблону, если выполняются следующие условия:
1. Анализ образца успешно завершился (то есть все стадии анализа имеют статус "*Завершено*").
2. Стадии "*Выравнивание*" и "*Выявление вариации числа копий*" включены в анализ образца и успешно завершились.
3. Шаблон отчета активен.
4. Шаблон отчета был добавлен в Систему до того, как была завершена обработка образца.

### Добавление шаблона отчета НИПТ {#add-report}

Чтобы добавить шаблон отчета НИПТ в систему, вы можете либо импортировать его,
либо создать самостоятельно.

#### Импорт шаблона отчета НИПТ {#import-report}

Скачайте файл шаблона (<a target="_blank" href="/reports/Отчет НИПТ.json" download="Отчет НИПТ.json">Отчет НИПТ</a>)
и импортируйте его в Систему, как описано [здесь](/settings/report-templates/report-templates#import-template).

#### Создание шаблона отчета НИПТ {#create-report}

Чтобы создать шаблон самостоятельно, выполните следующие действия:

1. На странице "*Шаблоны отчетов*" нажмите на <img src={require('/img/rus/add_template.png').default} width="170"/>,
чтобы открыть страницу добавления шаблона отчета.

2. На открывшейся странице заполните следующие поля:

- **Имя**: ```Отчет НИПТ```
- **Активен**: да

3. Нажмите <img src={require('/img/report_templates/save_temaplate.png').default} width="140"/>, чтобы сохранить шаблон.

4. Добавьте в шаблон все блоки, перечисленные ниже. Список блоков приведен в том порядке, в котором они идут в
шаблоне. Если никаких уточнений после названия блока нет, значит используется блок, формируемый по умолчанию.

#### Блоки шаблона: {#report-blocks}

1. **"Раздел"**:

*Имя*: ```Результат неинвазивного пренатального скрининга NIPT Helicon```

2. **"Информация об образце"**:

*Поля*:
- "Patient id";
- "First name";
- "Last name";
- "Date of birth";
- "Pregnancy month";
- "Age";
- "Middle name".

3. **"Информация об образце"**.
4. **"НИПТ"**:

*Имя*: ```Результаты исследования```

5. **"Дисклеймер"**:

*Текст*:
```
Неинвазивный пренатальный ДНК-скрининг (НИПС) предназначен для выявления женщин с высоким риском трисомий 21,13,18 хромосом и анеуплоидий по половым хромосомам (далее риск) у плода или высоким риском трисомий 21,13,18 хромосом у одного или обоих плодов при двуплодной беременности. НИПС не является диагностическим методом и не заменяет диагностические инвазивные тесты. Применяется с 10 недели одноплодной или двуплодной беременностей (без конкретизации для какого из плодов выявлен риск).
Интерпретация результатов НИПС проводится специалистом с учетом всех доступных клинических и лабораторных данных и не проводится в отрыве от этих данных.

Результат НИПС может быть положительным (высокий риск), отрицательным (низкий риск), невалидным (расчет риска невозможен из-за низкой фетальной фракции внеклеточной ДНК плода или по другим причинам).

В случае положительного результата НИПС необходима консультация врача-генетика с решением вопроса о проведении подтверждающего диагностического исследования. Отрицательный результат НИПС не исключает вероятность наличия у плода как исследуемых трисомий 21,13,18 хромосом и анеуплоидий по половым хромосомам, так и любых других аномалий. Возможно получение ложноположительных и ложноотрицательных результатов: результаты НИПС, расходящиеся с проведенными в дальнейшем диагностическими исследованиями.

НИПС не предназначен для выявления микроделеций/микродупликаций, мозаицизма, полиплоидии, структурных аномалий хромосом, иных хромосомных аномалий, моногенных и других генетических заболеваний, не связанных с анеуплоидиями 21,13, 18, X, Y хромосом, а также не предназначен для прогнозирования течения и исхода беременности.

Предполагаемый хромосомный пол будущего ребенка определяется по желанию беременной при одноплодной беременности.

```

6. **"Место для подписи"**.
