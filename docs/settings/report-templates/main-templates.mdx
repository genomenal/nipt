# Шаблон отчёта НИПТ

В данном разделе описан шаблон [отчёта НИПТ](/nipt-report) - медицинского отчёта,
который используется для представления результатов неинвазивного пренатального ДНК-скрининга образцов.
Скорее всего, он был добавлен в ваш профиль администратором при регистрации в Системе. Если у вас нет такого
шаблона, вы можете обратиться к администратору или добавить его самостоятельно, как описано ниже.

### Условия построения отчёта {#report-conditions}

Отчёт НИПТ строится для образца по описываемому шаблону, если выполняются следующие условия:
1. Анализ образца успешно завершился (то есть все стадии анализа имеют статус "*Завершено*").
2. Стадии "*Выравнивание*" и "*Выявление вариации числа копий*" включены в анализ образца и успешно завершились.
3. Шаблон отчёта активен.
4. Шаблон отчёта был добавлен в Систему до того, как была завершена обработка образца.

### Добавление шаблона отчёта НИПТ {#add-report}

Чтобы добавить шаблон отчёта НИПТ в систему, вы можете либо импортировать его,
либо создать самостоятельно.

#### Импорт шаблона отчёта НИПТ {#import-report}

Скачайте файл шаблона (<a target="_blank" href="/reports/Отчёт НИПТ.json" download="Отчёт НИПТ.json">Отчёт НИПТ</a>)
и импортируйте его в Систему, как описано [здесь](/settings/report-templates/report-templates#import-template).

#### Создание шаблона отчёта НИПТ {#create-report}

Чтобы создать шаблон самостоятельно, выполните следующие действия:

1. На странице "*Шаблоны отчётов*" нажмите на <img src={require('/img/rus/add_template.png').default} width="170"/>,
чтобы открыть страницу добавления шаблона отчёта.

2. На открывшейся странице заполните следующие поля:

- **Имя**: ```Отчёт НИПТ```
- **Активен**: да

3. Нажмите <img src={require('/img/report_templates/save_temaplate.png').default} width="140"/>, чтобы сохранить шаблон.

4. Добавьте в шаблон все блоки, перечисленные ниже. Список блоков приведен в том порядке, в котором они идут в
шаблоне. Если никаких уточнений после названия блока нет, значит используется блок, формируемый по умолчанию.

#### Блоки шаблона: {#report-blocks}

1. **"Раздел"**:

*Имя*: ```Результат неинвазивного пренатального скрининга NIPT Helicon```

2. **"Информация об образце"**:

*Поля*:
- "Patient id";
- "First name";
- "Last name";
- "Middle name";
- "Date of birth";
- "Age";
- "Gestational age";
- "Hospital";
- "Doctor name".

3. **"Информация об образце"** (поля по умолчанию).
4. **"НИПТ"**:

*Имя*: ```Результаты исследования```

5. **"Дисклеймер"**:

*Текст*:
```
Неинвазивный пренатальный ДНК-скрининг (НИПС) предназначен для выявления женщин с высоким риском трисомий 21, 13, 18 хромосом у плода при одноплодной беременности.

Риски анеуплоидий по половым хромосомам у плода при одноплодной беременности, а также риски трисомий 21, 13, 18 хромосом у плодов при двуплодной беременности (без конкретизации, у какого из плодов выявлен риск) приводятся в ознакомительных целях, могут быть информативными, но не являются достоверными в силу ограничений доступности данных для расчета точности.

НИПС не является диагностическим методом и не заменяет комбинированный пренатальный скрининг, диагностические инвазивные тесты и любые другие обследования, назначенные врачом во время беременности. Применяется с 10+0 недель беременности.

Интерпретация результатов НИПС проводится специалистом с учетом всех доступных клинических и лабораторных данных и не проводится в отрыве от этих данных.

Результат НИПС может быть положительным (высокий риск), отрицательным (низкий риск), невалидным (расчет риска невозможен из-за низкой фетальной фракции внеклеточной ДНК плода или по другим причинам).

В случае положительного результата НИПС необходима консультация врача-генетика с решением вопроса о проведении подтверждающего диагностического исследования. Отрицательный результат НИПС не исключает вероятность наличия у плода как исследуемых трисомий 21, 13, 18 хромосом, так и любых других аномалий. Возможно получение ложноположительных и ложноотрицательных результатов: результатов НИПС, расходящихся с проведенными в дальнейшем диагностическими исследованиями.

В отчёте приводится значение фракции фетальной ДНК - доли бесклеточной ДНК плода, присутствующей в кровотоке матери во время беременности. Нормальными считаются значения фетальной фракции, превышающие 4%.

НИПС не предназначен для выявления микроделеций/микродупликаций, мозаицизма, полиплоидии, структурных аномалий хромосом, иных хромосомных аномалий, моногенных и других генетических заболеваний, не связанных с анеуплоидиями 21, 13, 18 хромосом, а также не предназначен для прогнозирования течения и исхода беременности.

НИПС не рекомендуется назначать в силу низкой информативности и недостаточной для скрининга точности: беременным с многоплодной беременностью; беременным с самопроизвольной редукцией одного из плодов; один из биологических родителей эмбриона имеет аномальный кариотип; беременная получала аллогенное переливание крови, трансплантацию и аллогенную клеточную терапию в течение 1 года до наступления беременности; беременным со злокачественными новообразованиями.

НИПС не проводится: при наличии врожденных пороков развития у плода по данным УЗИ; присутствуют показания к инвазивной пренатальной диагностике; известны семейная история или высокий риск наследственных заболеваний у плода; при предоставлении беременной женщиной неполной, вводящей в заблуждение или недостоверной информации; другие условия, которые, по мнению врача, могут повлиять на точность результатов.

ИМТ>40, наступление беременности в цикле ЭКО снижают точность рассчитываемого риска: в этих случаях результат тестирования используется в ознакомительных целях.

Предполагаемый хромосомный пол будущего ребенка определяется по желанию беременной женщины при одноплодной беременности.

```

6. **"Место для подписи"**.
