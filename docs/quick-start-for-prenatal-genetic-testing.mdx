# Быстрый старт для пренатального генетического тестирования

Если вы хотите провести пренатальное генетическое тестирование с помощью Genomenal, то этот раздел для вас.

### 1. Создайте пациента {#create-patient}

import Login from '/docs/imported_components/_login.mdx';

<Login />

import PatientInfo from '/docs/imported_components/_patient_def.mdx';

Чтобы создать карточку пациента, нажмите
на <img src={require('/img/rus/add_patient_btn.png').default} width="150"/> на странице "Образцы":

import PatientPage from '/docs/imported_components/_patient_page.mdx';

<PatientPage />

<p align="center">
<img src={require('/img/rus/3a-patient-page.png').default} width="1000"/>
</p>

### 2. Загрузите образцы {#upload-samples}

Выберите образец или образцы для анализа.
В простейшем случае, образец - это результат секвенирования парных прочтений в формате FASTQ.

import UploadFiles from '/docs/imported_components/_upload_files_from_different_sources.mdx';

import FillCard from '/docs/imported_components/_fill_out_patient_card.mdx';

<p align="center">

</p>

### 3. Запустите анализ {#start-analysis}

Нажмите на <img src={require('/img/rus/start_analysis_btn.png').default} width="200"/>,
чтобы запустить анализ образцов.

Вы увидите [страницу пациента](/add-patient#patient-page) с добавленными образцами:

<p align="center">
<img src={require('/img/rus/analysis_started_prenatal.png').default} width="1000"/>
</p>

import DangerUploading from '/docs/imported_components/_danger_while_uploading.mdx';

<DangerUploading />

Вы можете определить, что анализ образца успешно завершен, если все статусы стадий окрашены зеленым или оранжевым:

<p align="center">
<img src={require('/img/rus/7a-stage-statuses.png').default} width="150"/>
</p>

## 4. Результаты анализа {#analysis-results}

Когда анализ завершен, вы можете посмотреть результаты. Откройте страницу образца, нажав на строку образца:

<p align="center">
<img src={require('/img/rus/7b-sample-row.png').default} width="900"/>
</p>

Затем нажмите на вкладку **Отчёт по CNV** и ознакомьтесь с отчётом:

<p align="center">
<img src={require('/img/rus/33-cnv-report-chr21.png').default} width="1000"/>
</p>

import SampleCNVReport from '/docs/imported_components/_sample_cnv_report.mdx';

<SampleCNVReport />

Отчёт можно экспортировать в PDF.

Кроме того, вы можете посмотреть результаты анализа для всех образцов пациента. То, как это сделать,
описано [здесь](/results/cnv-report/patient-cnv-report).

Дополнительно, [здесь](/sample-examples/normal-no-cnvs) вы можете найти примеры результатов пренатального
генетического тестирования образцов с различным качеством данных.
