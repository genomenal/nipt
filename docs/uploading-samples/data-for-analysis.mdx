# Образцы для анализа

Выберите образец или образцы для анализа.

NIPT helicon поддерживает файлы с прочтениями в формате **FASTQ**, полученные в результате секвенирования
свободно-циркулирующей ДНК (сцДНК) плазмы крови.
Образец - это либо ***один файл*** (в случае секвенирования одиночных прочтений), либо ***пара файлов***
(в случае секвенирования парных прочтений).

:::note На заметку
NIPT helicon умеет распознавать заархивированные файлы (GZIP, ZIP, BZIP2, 7-ZIP, XZ, WIM, RAR).
:::
