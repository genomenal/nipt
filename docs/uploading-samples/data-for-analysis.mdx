# Образцы для анализа

Выберите образец или образцы для анализа.

NIPT helicon поддерживает файлы в формате **FASTQ** с прочтениями, полученными в результате секвенирования
свободно-циркулирующей ДНК (сцДНК) плазмы крови. Файл должен содержать не менее 10 млн прочтений, из которых
не менее 90% картируются на геном. Файлы формата FASTQ поддерживаются как в неархивированном виде,
так и в архивах форматов GZ (GZIP), ZIP и BZ2 (BZIP2).
