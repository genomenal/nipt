# Загрузка образцов в запуск

Создав новый запуск, как описано [здесь](/nipt/runs#add-run), вы окажетесь на странице загрузки образцов в запуск:

### Настройки анализа {#workflow-settings}

Выберите нужную [настройку анализа](/settings/settings/settings) образцов.
Она будет применена ко всем образцам запуска.
Для выбора доступны настройки, добавленные в Систему администратором Программы.
Настройки влияют на параметры контроля качества, включение в анализ выравнивания и его параметры, а также
включение в анализ выявления вариации числа копий. Настройка "*По умолчанию*" включает все
шаги, необходимые для проведения неинвазивного пренатального тестирования.
После старта анализа его параметры изменить невозможно.

### Уровень НИПТ {#nipt-level}

Эта настройка регулирует, на каком уровне будет проведено неинвазивное пренатальное
тестирование (НИПТ) всех образцов запуска:
- *НИПТ T21* - неинвазивное пренатальное тестирование, направленный на выявление у плода трисомии по хромосоме 21.
- *НИПТ 3* - неинвазивное пренатальное тестирование наиболее частых аутосомных анеуплоидий:
трисомий по хромосомам 13, 18 и 21.
- *НИПТ 5* (выбран по умолчанию) - неинвазивное пренатальное тестирование,
направленное на выявление у плода наиболее частых аутосомных анеуплоидий: трисомий по хромосомам 13, 18 и 21,
а также половых анеуплоидий: моносомии X-хромосомы и дополнительной X-хромосомы.
Обратите внимание, что точность определения половых анеуплоидий не оценивалась и информация по ним может
быть использована только с целью ознакомления, но не в медицинских целях.

Обратите внимание, что оценивалась только точность определения трисомий 13, 18, 21 хромосом. Информация по
остальным анеуплоидиям может быть использована только с целью ознакомления, но не в медицинских целях.

Этот параметр влияет на отображение результатов в колонке "*Результат*" и
в [отчёте НИПТ](/nipt-report) каждого образца запуска.

### Загрузка образцов {#upload-samples}

Загрузите выбранные образцы одним из предложенных ниже способов. Количество образцов в запуске не ограничено.
Требования к образцам описаны [здесь](/nipt/uploading-samples/data-for-analysis).

#### Показывать пол плода {#fetus-sex}

По умолчанию предполагаемый пол плода, определённый в результате исследования, отображается
в [отчёте НИПТ](/nipt-report). Эту опцию можно отключить для любого образца, загружаемого в запуск.

#### Двуплодная беременность {#multiple-pregnancy}

Если вы добавили в запуск образец, полученный в результате секвенирования свободно-циркулирующей ДНК плазмы
крови женщины, беременной двумя плодами, включите для него соответствующую опцию. Риски трисомий 21, 13, 18
хромосом у плодов при двуплодной беременности (без конкретизации, у какого из плодов
выявлен риск) приводятся в ознакомительных целях, могут быть информативными, но не являются достоверными в силу
ограничений доступности данных для расчета точности.
При активации параметра "*Двуплодная беременность*" параметр "*Показывать пол плода*" становится неактивным.
