# Загрузка дополнительных образцов для пациента

Если вы хотите добавить для пациента образцы вдобавок к
образцам, [загруженным ранее](/nipt/uploading-samples/upload-for-new-patient),
нажмите на кнопку <img src={require('/img/rus/add_samples_to_patient.png').default} width="160"/>
на [странице пациента](/nipt/add-patient#patient-page). Откроется окно загрузки дополнительных образцов:

<p align="center">
<img src={require('/img/rus/upload_window.png').default} width="600"/>
</p>

### Загрузка образцов {#upload-samples}

Загрузите выбранные образцы одним из предложенных ниже способов.
Требования к образцам описаны [здесь](/nipt/uploading-samples/data-for-analysis).
Можно загрузить сразу несколько
образцов одного пациента, например, если у вас имеются технические реплики.

import UploadFiles from '/docs/imported_components/_upload_files_from_different_sources.mdx';

<UploadFiles />

Загруженные по ошибке образцы можно удалить, наведя курсор на их карточку и
нажав на <img src={require('/img/rus/delete_btn.png').default} width="20"/>.

### Настройки анализа {#analysis-settings}

Выберите нужную [настройку анализа](/settings/settings/settings) загружаемых образцов.
Настройку, выбранную по умолчанию, можно поменять, как
описано [здесь](/nipt/settings/settings/settings#set-preset-as-default).
Настройки влияют на параметры контроля качества, включение в анализ выравнивания и его параметры, а также
включение в анализ выявления вариации числа копий. После начала анализа его параметры изменить
невозможно.

### Уровень НИПТ {#nipt-level}

Для загружаемого образца или образцов можно выбрать, на каком уровне будет проведено неинвазивное пренатальное
тестирование (НИПТ):
- *НИПТ T21* - неинвазивный пренатальный скрининг, направленный на выявление у плода трисомии по хромосоме 21.
- *НИПТ 3* - неинвазивный пренатальный скрининг наиболее частых аутосомных анеуплоидий: трисомии по хромосоме 13
(синдром Патау), трисомии по хромосоме 18 (синдром Эдвардса), трисомии по хромосоме 21.
- *НИПТ 5* (выбран по умолчанию) - неинвазивный пренатальный скрининг, направленный на выявление у плода
наиболее частых аутосомных анеуплоидий: трисомии по хромосоме 13 (синдром Патау), трисомии по хромосоме 18
(синдром Эдвардса), трисомии по хромосоме 21, а также половых анеуплоидий: дисомии Х-хромосомы (XXY; синдром
Клайнфельтера) и моносомии X-хромосомы (X0; синдром Шерешевского-Тёрнера). Обратите внимание, что точность
определения половых анеуплоидий не оценивалась и информация по ним может
быть использована только с целью ознакомления, но не в медицинских целях.

Этот параметр повлияет на отображение результатов в [медицинском отчёте](/nipt-report).

### Пол плода {#fetus-sex}

По умолчанию предполагаемый пол плода, определённый в результате исследования, отображается
в [медицинском отчёте](/nipt-report). Эту опцию можно отключить для загружаемого образца.

### Двуплодная беременность {#multiple-pregnancy}

Если вы хотите проанализировать образец, полученный в результате секвенирования свободно-циркулирующей ДНК плазмы
крови женщины, беременной двумя плодами, включите для него соответствующую опцию. Риски трисомий 21, 13, 18
хромосом у плодов при двуплодной беременности (без конкретизации, у какого из плодов
выявлен риск) приводятся в ознакомительных целях, могут быть информативными, но не являются достоверными в силу
ограничений доступности данных для расчета точности.

### Добавление образцов {#add-samples}

Перетащите загруженный образец из раздела "*Образцы без категорий*" слева в окно добавления
или нажмите на загруженный образец, а затем на окно:

<p align="center">
<img src={require('/img/rus/add_sample_window.png').default} width="300"/>
</p>

После этого под карточкой добавленного образца снова появится такое окно.
Если вы хотите загрузить несколько образцов для пациента, перетащите их один за другим в окно добавления.

Чтобы вернуть образец в раздел "*Образцы без категории*",
наведите курсор на образец и нажмите на крестик <img src={require('/img/rus/remove.png').default} width="20"/>.

### Запуск анализа {#run-analysis}

Нажмите на <img src={require('/img/rus/start_analysis_from_modal.png').default} width="160"/>, чтобы начать
анализ образцов.

import DangerUploading from '/docs/imported_components/_danger_while_uploading.mdx';

<DangerUploading />

Отслеживать процесс анализа образца можно на [его странице](/results#workflow) или
посредством [оповещений о завершении его обработки](/settings/profile#notification).
