# Настройка параметров анализа




Если вы используете образец здоровой ткани для поиска ***наследственных мутаций, структурных вариаций и
вариаций числа копий***, выберите настройку анализа
**[Germline variant analysis](/settings/settings/main-presets#germline-variant-analysis)**:

![Настройка параметров анализа](/img/version_1.02/germline_preset.png)

Если вы используете образец здоровой ткани для ***пренатального генетического тестирования***
(поиск вариаций числа копий), выберите настройку анализа
**[Prenatal analysis](/settings/settings/main-presets#prenatal-analysis)**:

<img src={require('/img/version_1.02/prenatal_preset.png').default} width="600"/>
</p>
