# Неинвазивный пренатальный скрининг с помощью NIPT Helicon

Программа NIPT Helicon направлена на проведение неинвазивного пренатального тестирования (НИПТ).
Она позволяет быстро обработать данные секвенирования свободно-циркулирующей ДНК (сцДНК)
плазмы крови, получить информацию о вариациях числа копий ДНК, а также
предоставляет инструменты для удобной и качественной интерпретации результатов. Далее вы найдете информацию о том,
как загрузить ваши данные, о процессе анализа и доступе к его результатам.
