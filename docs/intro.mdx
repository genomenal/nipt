# Программное обеспечение для обработки данных неинвазивного пренатального ДНК-скрининга анеуплоидий плода по крови матери

Данное программное обеспечение позволяет получить прогностическую информацию о риске трисомий 21,
13, 18 хромосом в образцах свободно-циркулирующей ДНК (сцДНК) плазмы крови беременных женщин.
Эта информация используется специалистами для оценки необходимости проведения
подтверждающего диагностического исследования. Кроме того, в качестве дополнительной информации
выдаются анеуплоидии и микроделеции по другим аутосомам и половым хромосомам X и Y у плода. Эта информация может быть
использована с целью ознакомления, но не может быть использована в медицинских целях.

Программа позволяет быстро обработать данные секвенирования свободно-циркулирующей ДНК
плазмы крови, получить информацию о вариациях числа копий ДНК, а также
предоставляет инструменты для удобной и качественной интерпретации результатов неинвазивного пренатального
ДНК-скрининга. Далее вы найдете информацию о том, как загрузить ваши данные, о процессе анализа и доступе
к его результатам.
