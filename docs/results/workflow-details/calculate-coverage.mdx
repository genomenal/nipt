# Вычисление покрытия

На данном этапе рассчитывается покрытие генов и целевых участков
выравниваниями прочтений образца. Эти показатели важны для оценки качества картирования в целом и
на интересующих вас областях (генах, транскриптах).
Вычисление покрытия запускается после успешного выполнения стадии
["Выравнивание"](/results/workflow-details/fastq-analysis/variant-discovery-preprocessing).

<p align="center">
<img src={require('/img/rus/calculate_coverage.png').default} width="700"/>
</p>

Стадия анализа образца "Вычисление покрытия" включает одноименную задачу. Ход выполнения задачи следующий:
- [bedtools coverage](https://bedtools.readthedocs.io/en/latest/content/tools/coverage.html)
вычисляет глубину и широту покрытия выравниваний прочтений образца
экзонами из [Ensembl](https://www.ensembl.org/index.html) и
строит гистограмму покрытия для каждого экзона и сводную гистограмму для всех экзонов.
- [bedtools genomecov](https://bedtools.readthedocs.io/en/latest/content/tools/genomecov.html)
вычисляет гистограммы покрытия выровненных последовательностей образца на референсный геном.
