# Вариации числа копий

Секция "***Вариации числа копий***" содержит результаты
выявления участков изменения числа копий (copy number variations; CNVs)
и расчёта значений копийности для данного образца на фоне других образцов.

<p align="center">
<img src={require('/img/results/germline_cnvs.png').default} width="800"/>
</p>

Результаты выявления вариаций числа копий представлены в следующих форматах:
1. **Полногеномный график копийности** в формате JPG. Чтобы открыть график,
нажмите на <img src={require('/img/rus/25-plot_file.png').default} width="45"/>. Пример графика:

<p align="center">
<img src={require('/img/rus/28-genome-wide-plot.png').default} width="1000"/>
</p>

На горизонтальной шкале мы видим хромосомы, по вертикали - копийность хромосомы.
На графике мы видим, что для этого образца копийность хромосомы 21 увеличена в несколько раз.

2. **Статистика по хромосомам** в формате TSV - файл,
удобный для рассмотрения изменения копийности на хромосомах.
Чтобы скачать файл, нажмите на <img src={require('/img/results/cnv_tsv.png').default} width="45"/>.

3. **Статистика по бинам** в формате TSV - файл,
удобный для рассмотрения изменения копийности в бинах. Бины - это небольшие интервалы, в которых проводится
подсчёт количества прочтений
и которые создаются в результате разделения референсного генома на интервалы равного размера.
Чтобы скачать файл, нажмите на <img src={require('/img/results/cnv_tsv.png').default} width="45"/>.

4. **Статистика по сегментам** в формате TSV - файл,
удобный для рассмотрения изменения копийности в сегментах. Сегменты - это крупные области генома,
в которых проводится подсчёт количества прочтений и которые получаются в результате объединения соседних бинов
со схожей копийностью.
Чтобы скачать файл, нажмите на <img src={require('/img/results/cnv_tsv.png').default} width="45"/>.
