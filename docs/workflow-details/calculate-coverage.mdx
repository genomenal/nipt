# Вычисление покрытия

На данном этапе рассчитывается понуклеотидное покрытие выровненных последовательностей образца на референсный геном.
Этот показатель важен для оценки качества картирования. Результат вычисления представлен
в ["Биоинформатическом отчёте"](/bioinformatic-report#coverage-report).
Вычисление покрытия запускается после успешного выполнения стадии ["Выравнивание"](/workflow-details/alignment),
если в анализ образца включена
стадия [выявления вариации числа копий](/settings/settings/cnv-discovery-settings).

<p align="center">
<img src={require('/img/rus/calculate_coverage.png').default} width="600"/>
</p>

Стадия анализа образца "Вычисление покрытия" включает одноименную задачу. Во время её выполнения
[bedtools genomecov](https://bedtools.readthedocs.io/en/latest/content/tools/genomecov.html)
вычисляет гистограммы покрытия выровненных последовательностей образца на референсный геном.
