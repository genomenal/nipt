# Выявление вариации числа копий

После успешного выполнения
стадии ["Выравнивание"](/workflow-details/alignment)
для образца может запуститься выявление вариации числа копий (copy number variations, CNVs), во время которого
определяется, сколько копий участка ДНК было в среднем представлено в секвенированном геноме.
Для запуска стадии необходимо, чтобы она была [включена](/settings/settings/cnv-discovery-settings) в
анализ. Если выявление вариации числа копий исключено из анализа, то после успешного выполнения
стадии ["Выравнивание"](/workflow-details/alignment) анализ завершается.
При ошибке выполнения любой из перечисленных ниже задач анализ образца останавливается.

<p align="center">
<img src={require('/img/rus/cnv_pipeline.png').default} width="650"/>
</p>

Стадия анализа образца "Выявление вариации числа копий" включает следующие задачи:
1. **Предварительная обработка для выявления вариации числа копий**.
В ходе задачи происходит извлечение информации о покрытии из файла выравнивания.
Для дальнейшего анализа берутся только
картированные прочтения с MAPQ ≥ 1. MAPQ (MAPping Quality) - это вероятность, равная −10 log10 P, округлённому
до целого числа, где P - это вероятность того, что выравнивание неверно.
Получившийся файл с собранным профилем покрытия можно скачать в разделе "*Файлы с результатами*" в деталях задачи
"*Предварительная обработка для выявления вариации числа копий*" ("*Скачать H5*").
2. **Выявление вариации числа копий** - анализ образца против референса с целью выявления хромосомных анеуплоидий.
Референсный файл натренирован на 500 образцах.
Результатом выполнения задачи являются следующие файлы, которые можно скачать в разделе "*Файлы с результатами*"
в деталях задачи "*Выявление вариации числа копий*":
- Полногеномный график копийности в формате JPG ("*Открыть Plot all chr JPG*"). Пример графика:

<p align="center">
<img src={require('/img/rus/28-genome-wide-plot.png').default} width="1000"/>
</p>

На горизонтальной шкале располагаются хромосомы, на вертикальной шкале - их копийность.
На приведённых графиках мы видим, что для данного образца копийность хромосомы 21 увеличена в несколько раз.

- Двумерный график пространства главных компонент в формате PNG ("*Открыть Plot PCA PNG*"). Пример графика:

<p align="center">
<img src={require('/img/rus/pca.png').default} width="1000"/>
</p>

- Итоговый результат НИПС в формате JSON ("*Скачать Summary result JSON*"). Содержит следующие данные:
   - *"gender"* - предсказанный пол плода;
   - *"FF"* - рассчитанное значение фракции фетальной ДНК;
   - информация по каждой анеуплодии:
       - *"name"* - название анеуплодии (*Tn* - трисомия по хромосоме *n*, *Mn* - моносомия по хромосоме *n*);
       - *"Zscore"* - оценка, представляющая собой значение рика НИПС. Описывает положение сырого результата с точки
       зрения его расстояния от среднего значения;
       - *"affected_chrom"* - затронутая хромосома;
       - *"direction"* - направление;
       - *"ratio"* - доля бесклеточной ДНК плода;
       - *"detected"* - задетектирована ли анеуплоидия в образце: true или false;
       - *"Mosaicism"* - доля мозаицизма;
   - *"pca_dst_median"* - среднее индекса межпопуляционного разнообразия;
   - *"Z_score_median_per_bin"* - среднее значение риска НИПС на бин;
   - *"Z_score_median_per_chrom"* - среднее значение риска НИПС на хромосому;
   - *"median_segment_variance"* - медианная дисперсия сегментов - медиана набора дисперсий, где каждая дисперсия
   соответствует дисперсии сегмента. Является мерой шума, обратно пропорциональна размеру бина и глубине
   прочтения;
   - *"mapped_reads"* - суммарное количество выравниваний прочтений образца на референсный геном;
   - *"read_after_filtering"* - количество выровненных прочтений в файле, оставшихся после фильтрации;
   - *"Y_frac"* - фракция Y-хромосомы.

- Статистика по бинам в формате TSV ("*Скачать Bins TSV*") - файл, удобный для рассмотрения изменения копийности
в бинах. Бины - это небольшие интервалы, в которых проводится подсчёт количества прочтений
и которые создаются в результате разделения референсного генома на интервалы равного размера.
- Статистика по сегментам в формате TSV ("*Скачать Segments TSV*") - файл, удобный для рассмотрения изменения
копийности в сегментах. Сегменты - это крупные области генома, в которых проводится подсчёт количества прочтений
и которые получаются в результате объединения соседних бинов со схожей копийностью.
- Статистика по хромосомам в формате TSV ("*Скачать Chrom stat TSV*") - файл,
удобный для рассмотрения изменения копийности на хромосомах.

После стадии "Выявление вариации числа копий" анализ может
продолжиться [генерацией отчётов](/workflow-details/report-generation).
